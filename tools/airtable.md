## Get

```python
from naas_drivers import airtable
api_key = "******"
db_key = "appuBFPzX94pEqXUJ"
table = "Opportunities"
view = "MyView" # the name of your view in airtable
data = airtable.connect(api_key, db_key, table).get(view=view, maxRecords=20)
```
