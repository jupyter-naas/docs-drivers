## Users

### Get

```python
import nass_drivers
api_key = "api_key"
naas_drivers.thinkific.connect(api_key).users.get(
    uid="uid"
)
```

### Send

```python
import nass_drivers
api_key = "api_key"
naas_drivers.thinkific.connect(api_key).users.send(
    email="bob@cashstory.com"
)
```

### Delete

```python

    uid="uid"
```
